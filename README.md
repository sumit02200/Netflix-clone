# Netflix Frontend Clone

A responsive Netflix UI Clone built using HTML, CSS, and JavaScript.
This project replicates the core design and layout of Netflix to demonstrate frontend development skills and UI/UX implementation.

# Key Features

• Responsive homepage design
• Hero banner with call-to-action
• Movie & series rows (slider layout)
• Hover effects & animations
• Navigation bar similar to Netflix
• Mobile-friendly layout
• Clean and scalable code structure
